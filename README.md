# 🚀 Skill Tracker & AI Career Readiness Platform

> Transform your skills into a **data-driven career strategy**.
> Analyze, track, and accelerate your journey toward high-demand tech roles using AI.

---

## ✨ Overview

A modern **full-stack AI-powered platform** that helps developers and professionals:

* 📊 Understand their current skill landscape
* 🎯 Measure readiness for target roles
* 🧠 Generate intelligent learning roadmaps
* 🚀 Systematically grow into industry-ready candidates

---

## 🌟 Core Features

### 🧩 Smart Skill Management

* Organize skills across domains: **Frontend, Backend, DevOps, Database**
* Track progression with meaningful stages:

  * 🧠 *Learned* (Theory)
  * 🛠️ *Practiced* (Projects)
  * 🚀 *Applied* (Real-world)
* Dynamic rating system with interactive UI controls

---

### 📄 AI Resume Intelligence

* Upload resumes for **instant skill extraction**
* Automatically enrich your skill profile
* Detect hidden or overlooked competencies

---

### 🎯 Career Readiness Engine

* Select a target role (e.g., *Frontend Developer, Backend Engineer*)
* Get:

  * 📊 **Skill match percentage**
  * ⚡ Strength & gap analysis
  * 🎯 Clear improvement insights

---

### 🧠 AI-Generated Learning Roadmaps

* Personalized, step-by-step growth plans
* Focused on **missing skills**
* Includes:

  * 📚 Courses
  * 🛠️ Projects
  * 🎯 Actionable milestones

---

### 💎 Premium UI Experience

* ⚡ Smooth animations with Framer Motion
* 🎨 Modern design system (light + dark themes)
* 📊 Interactive charts & analytics
* 🧩 Clean, scalable component architecture

---

## 🛠️ Tech Stack

| Layer                  | Technologies                                        |
| ---------------------- | --------------------------------------------------- |
| **Frontend**           | React 19, Tailwind CSS, Framer Motion, Lucide Icons |
| **Backend**            | Node.js, Express                                    |
| **AI Integration**     | Gemini AI                                           |
| **Data Visualization** | Chart.js, React Chart.js 2                          |
| **Build Tools**        | ESBuild, Vite                                       |

---

## 🏗️ Architecture

```bash
├── server.ts              # Express server + API layer
├── src/
│   ├── App.jsx            # Root layout & navigation
│   ├── main.jsx           # Entry point
│   ├── index.css          # Tailwind setup & global styles
│   ├── components/        # Reusable UI components
│   ├── pages/             # Application views
│   ├── contexts/          # Global state (auth, etc.)
│   └── services/          # API logic
```

## ⚡ Getting Started

### 1️⃣ Install dependencies

```bash
npm install
```

### 2️⃣ Setup environment

Create a `.env` file:

```env
PORT=3000
GEMINI_API_KEY=your_api_key_here
```

---

### 3️⃣ Run development server

```bash
npm run dev
```

App runs at:

```
http://localhost:3000
```

---

### 4️⃣ Build for production

```bash
npm run build
npm start
```
